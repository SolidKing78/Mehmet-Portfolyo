# Proje Ozetleri

Bu dosyada one cikan projelerin kisa ve detayli ozetleri bulunur.

## 1) Prefabrik & Hafif Celik CAD/CAM Platformu

- **Slug:** `prefabrik-hafif-celik-cadcam`
- **Alan:** CAD/CAM
- **Kisa Ozet:** C++ cekirdek uzerinde, 3B modelden roll-form uyumlu uretim verisine gecis.
- **Detaylar:**
  - AI-assisted / vibe coding yaklasimiyla modern urun mimarisi
  - Otomatik modelleme ve uretim verisi uretimi
  - Hata kontrol mekanizmalari
  - Olceklenebilir lisans yonetimi
- **Etki:**
  - Hazirlik sureclerinde yaklasik `%70` verimlilik bandi
  - Manuel adimlarda belirgin azalma

## 2) SolidWorks CAD Otomasyonu

- **Slug:** `solidworks-cad-otomasyonu`
- **Alan:** SolidWorks API
- **Kisa Ozet:** Gencer Otomotiv icin parametrik 3D tasarim + tek tus dokumantasyon.
- **Detaylar:**
  - SolidWorks API, VBA, .NET, C# ile otomasyon
  - Kisitli olcu girdisi ile komple montaj akisi
  - PDF/DWG/DXF ve bukum resimleri dahil paket cikti
- **Etki:**
  - Cikti standardizasyonu
  - Tekrarlayan modelleme yukunde azalma

## 3) ParametriX 2D -> 3D Converter

- **Slug:** `parametrix-2d-3d`
- **Alan:** Product Utility
- **Kisa Ozet:** 2B girdileri parametrik kurallarla 3B modele donusturen tekrar uretilebilir akis.
- **Detaylar:**
  - Geometri isleme + kural dogrulama katmani
  - Kullanici akisinda sadelesme
  - Manuel duzeltme ihtiyacinda azalma
- **Etki:**
  - Donusum kalitesinde ongorulebilirlik
  - Muhendislik tekrarlarinda tutarli sonuc
